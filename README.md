# Modern Glassmorphic Accordion Portfolio

A sleek, interactive, and high-performance interactive FAQ/About-Me single-page portfolio built with modern web standards. This project features an immersive glassmorphic design system accented by dynamic custom cursor tracking, animated backdrop-filter ambient orbs, fluid CSS micro-interactions, and full accessibility implementation.

## ✨ Features

- **Interactive Glassmorphic Interface:** Leverages high-blur backdrops, subtle transparency masking, and compound color variables to deliver an ultra-modern aesthetic.
- **Dynamic Cursor Glow:** A smooth, custom-engineered mouse tracker (`requestAnimationFrame` loop with linear interpolation) that guides user focus without causing visual lag.
- **Fluid Custom Accordion:** Fully responsive, animated FAQ system utilizing dynamic `scrollHeight` tracking to expand and collapse cleanly regardless of screen sizes.
- **Staggered Scroll-Driven Reveals:** An engineered `IntersectionObserver` system that tracks component entrances, revealing footer sections gracefully with a cubic-bezier transition framework.
- **Responsive Layout:** Adaptive styling down to mobile configurations (`480px` breakpoint optimization) for perfect text balancing and structural integrity.
- **Accessible Core (A11y):** Implements correct semantic tags (`main`, `footer`), explicit `aria-expanded` attributes, ARIA lists/listitems roles, and native keyboard navigation listeners (`Enter` / `Space` support).
- **Performance Optimized:** Includes custom `@media (prefers-reduced-motion: reduce)` rules for high accessibility compliance and lightweight assets.

## 🛠️ Tech Stack

- **Markup:** Semantic HTML5 Structure
- **Styling:** CSS3 Custom Properties (Variables), Nested Selectors, CSS Keyframe Animations
- **Interactivity:** Vanilla Javascript (ES6+), Web APIs (`IntersectionObserver`, `requestAnimationFrame`)
- **Typography & Icons:** Google Fonts (Outfit, Space Grotesk) & FontAwesome v6
